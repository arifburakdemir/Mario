# Mario

## Aim of Project
Devoloping Mario and 2D physics using JAVA and Swing for 2016 Collage Project. The main challenge of this prject was developing 2D platform game without using any game engine.

## GamePlay
Clasiccal mario gameplay. Jumping, runing, killing mobs, collecting coins, Eating Mushroms developed in SWING Java to experience and develop 2D physics.
